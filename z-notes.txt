1) package.json

The react package defines new React components. React isn't 
confined to the browser.

The react-dom package serves as the entry point to the DOM 
to render the JSX by using the virtual DOM.

2) index.js

You can think of the index.js file as the main entry point 
for the application.
The ReactDOM library is rendering the App component at the 
root element in the HTML.

3) index.HTML

The ReactDOM library is rendering the App component at the
 root element in the HTML.


1) 20.1.5

React

React components follow the PascalCase naming convention.

You need to import React in every component file. You can 
also import any images or CSS you want to use at the top.

JSX can represent HTML in JavaScript. In fact, the way 
React uses JSX behind the scenes is very similar to 
document.createElement().
